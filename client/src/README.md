# Hospital Management System

A full-stack Hospital Management System built using the MERN Stack.

## Features

- Login Authentication
- Dashboard
- Patient Management
- Doctor Management
- Appointment Management
- Search Patients
- Search Doctors
- Search Appointments
- Export to Excel
- Export to PDF
- Dashboard Statistics
- Charts
- Loading Spinner
- Responsive Design
- Bootstrap Icons
- 404 Page
- Footer

## Technologies Used

### Frontend
- React.js
- TypeScript
- Bootstrap
- Axios
- React Router
- Chart.js
- XLSX
- jsPDF

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
